## 1.分类

- 基本（值）类型
  - String：任意字符串
  - Number：任意数字
  - boolean：true/false
  - undefined：undefined
  - null：null
- 对象（引用）类型
  - Object：任意对象
  - Function：一种特别的对象（**可以执行**）
    - 普通对象是用来存数据的，不能执行，而函数是用来存代码的，可以执行。
  - Array：一种特别的对象（**数值下标，内部数据是有序的**）

## 2.类型判断

- typeof
  - 可以判断undefined / 数字 / 字符串 / 布尔值
  - 不能判断null和object，object和array
- instanceof
  - 判断**对象**的**具体类型**
- === / == （尽量不要用两个等号）
  - ===可以判断undefined，null（五种基本类型中可以判断null）

```js
//基本类型判断
var a
console.log(a) // undefined
console.log(typeof a) // 'undefined',typeof返回数据类型的字符串表达
console.log(undefined === 'undefined') // false
console.log(a === undefined) //true

a = null
console.log(typeof a) // 'object'
console.log(a===null) //true
```

```js
//对象类型判断
var b1 = {
	b2: [1, 'abc', console.log],
	b3: function () {
		console.log('b3')
        return function() {
            return 'cjj'
        }
	}
}

console.log(b1 instanceof Object)//判断b1是不是构造函数的实例 true
console.log(b1 instanceof Array)//false
console.log(b1.b2 instanceof Array, b1.b2 instanceof Object)//true true
console.log(b1.b3 instanceof Function, b1.b3 instanceof Object)//true true

console.log(typeof b1.b3 === 'function')// true
console.log(typeof b1.b2[2] === 'function')//true
b1.b2[2](4) //判断函数是否执行，就看它有没有() //4
console.log(b1.b3()())//cjj
```

## 4.undefined与null的区别？

```js
var a
console.log(a)
a = null
console.log(a) //null
//答：undefined代表定义，但未赋值；null定义并赋值了，只是值为null
//思考：那什么时候赋值为null呢？
var b = null //初始赋值为null，表明将要赋值为对象//想要定义一个b的对象，但暂时还不知道里面的值，就可以先设为null
var b = ['cjj', 12]
b = null//让b指向的对象成为垃圾对象（被垃圾回收器回收）
//答：1.初始赋值对象 2.让对象成为垃圾对象
```
